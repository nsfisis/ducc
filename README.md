# Ducc

![Ducc logo](./assets/logo.svg)

Ducc is a toy C compiler, based on [my another tiny C compiler](https://github.com/nsfisis/P4Dcc).


## Dependencies

* gcc
* make
* [just](https://github.com/casey/just)


## Build

```
$ just build
```

```
$ just build-upto-5-gen
```


## Test

```
$ just test
```

```
$ just test-self-hosted
```
