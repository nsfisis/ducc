# Ducc

![Ducc logo](./assets/logo.svg)

Ducc, Decidedly Unimplemented C compiler, is a toy C compiler.


## Dependencies

* gcc
* make
* [just](https://github.com/casey/just)


## Build

```
$ just build
```

```
$ just build-upto-5-gen
```


## Test

```
$ just test
```

```
$ just test-self-hosted
```
