# Ducc

Ducc is a toy C compiler, based on [my another tiny C compiler](https://github.com/nsfisis/P4Dcc).


## Dependencies

* gcc
* [just](https://github.com/casey/just), a general-purpose task runner


## Build

```
$ just build
```


## Test

```
$ just test
```
